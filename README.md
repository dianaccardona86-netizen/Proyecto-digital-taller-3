# Proyecto-digital-taller-3
Este proyecto es realizado en microphyton y en codigo HTML, junto con un montaje fisico que contiene un potenciometro, tres led , sensor LM35,  resistencias, una fotoresistencia, ESP32, entre otros componentes. Los codigos al ejecutarse en conjunto adquieren temperatura, temperatura de referencia y luminosidad, que se  visualizan en una pagina web.
